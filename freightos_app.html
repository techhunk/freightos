Connect to Google Sheets (Free Backend)

Paste your Google Apps Script Web App URL below to sync all data to Google Sheets. Your data will be accessible from any device — phone or PC — instantly.

📋 Step-by-Step Setup Guide
Step 1. Go to sheets.google.com → Create a new blank spreadsheet → Name it "FreightOS Data"
Step 2. In the spreadsheet, click Extensions → Apps Script
Step 3. Delete all existing code. Paste the entire contents of apps_script_backend.js (the second file you downloaded)
Step 4. Click Run → setupSpreadsheet. Allow permissions when asked. Check the Logs — copy your Spreadsheet ID shown there. Replace YOUR_SPREADSHEET_ID_HERE at the top of the script with it.
Step 5. Click Deploy → New Deployment → Web App. Set Execute as: Me, Who has access: Anyone. Click Deploy. Copy the Web App URL.
Step 6. Paste that URL above and click Connect. Done — your app now syncs to Google Sheets! 🎉
📱 Install on Android Phone

Since this is a file, you have two options to use it on mobile:

Option A — Host it free on GitHub Pages: Upload this HTML file to a GitHub repository, enable Pages, and open the URL on your phone in Chrome → tap menu → "Add to Home Screen". Works like an app!
Option B — Local only: Copy the HTML file to your phone, open with Chrome → "Add to Home Screen". Data stays on phone only (no sync) unless connected to Google Sheets.
